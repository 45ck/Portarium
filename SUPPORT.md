# Support

Use the channel that matches the kind of help you need.

## Questions And Ideas

Use [GitHub Discussions](https://github.com/45ck/Portarium/discussions) for:

- setup questions
- architecture questions
- agent integration ideas
- roadmap discussion
- help understanding docs

## Bugs And Feature Requests

Use [GitHub Issues](https://github.com/45ck/Portarium/issues/new/choose) for:

- reproducible bugs
- missing documentation
- focused feature requests
- test failures with logs and reproduction steps

## Security Issues

Do not report vulnerabilities in public issues.

Follow [SECURITY.md](SECURITY.md) and use GitHub private vulnerability reporting.
