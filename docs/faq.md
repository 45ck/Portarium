# FAQ

## What is Portarium?

Portarium is the product name for an open-source multi-tenant control plane for governable operations.

## What does VAOP mean?

VAOP is the internal architecture acronym used in technical and domain contexts.

## Is this production-ready?

Not yet. The project is active, with contract-first APIs and runnable scaffolds while adapters and persistence continue to mature.

## Where is the API contract?

`docs/spec/openapi/portarium-control-plane.v1.yaml`

## How do I run it locally?

See `docs/getting-started/local-dev.md`.
